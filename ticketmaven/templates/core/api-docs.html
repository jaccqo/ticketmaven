API Documentation

API Home
User Profiles
  • GET /api/user-profiles/ - List all user profiles
  • POST /api/user-profiles/ - Create a new user profile
  • GET /api/user-profiles/{id}/ - Retrieve a user profile by ID
  • PUT /api/user-profiles/{id}/ - Update a user profile by ID
  • DELETE /api/user-profiles/{id}/ - Delete a user profile by ID

Note: Token authentication is required for these endpoints.

Example: Retrieve user profile with ID 1: GET /api/user-profiles/1/

Activities
  • GET /api/activities/ - List all activities
  • POST /api/activities/ - Create a new activity
  • GET /api/activities/{id}/ - Retrieve an activity by ID
  • PUT /api/activities/{id}/ - Update an activity by ID
  • DELETE /api/activities/{id}/ - Delete an activity by ID

Note: Token authentication is required for these endpoints.

Example: Retrieve activity with ID 1: GET /api/activities/1/
